Sobre a Loja

O Bazaar do Artífice é uma loja fictícia voltada para a venda de itens de RPG, como poções, pergaminhos e artefatos mágicos.
Este projeto foi desenvolvido com HTML, CSS e JavaScript para fins de aprendizado e demonstração de layout responsivo e interativo.
